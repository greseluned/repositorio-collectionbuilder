Créditos técnicos - CollectionBuilder

Esta colección digital está construida con CollectionBuilder, un framework de código abierto para la creación de sitios web de colecciones y exposiciones digitales, desarrollado por bibliotecarios de la Universidad de Idaho siguiendo la metodología Lib-Static.

Utilizando la plantilla CollectionBuilder-CSV y el generador de sitios estáticos Jekyll, este proyecto crea una interfaz atractiva de exploración basada en metadatos.
